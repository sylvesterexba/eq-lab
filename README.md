# Interactive EQ Lab

Interactive EQ Lab is a visual EQ learning tool for practicing frequency, gain, Q, and filter type decisions.

互動 EQ 實驗室是一個視覺化 EQ 學習工具，用來練習 Frequency、Gain、Q 與 Filter Type 的操作判斷。

## Features

- Frequency Atlas
- EQ Response Graph
- Interactive Knobs
- EQ Type Controls
- Floating Summary
- Learning Accordion
- Mobile Friendly
